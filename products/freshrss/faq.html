What apps do you recommend for reading {{ product.name }}?

For Android users, we recommend Capy Reader. For Linux desktop users, NewsFlash is an excellent choice. Both apps work great with {{ product.name }} and offer a smooth reading experience. Any compatible app should work as expected. You can also refer to the official list of {{ product.name }} compatible apps for other platforms such as Mac, iOS, and Windows.

How do I set up mobile app access?

To use {{ product.name }} with mobile apps, you'll need to set up an API password, which is separate from your main user password. After logging in, go to your settings and set an API password. You'll use this API password (not your regular login password) when connecting through mobile apps.

How long does account creation take?

In most cases, your account will be created quickly after your subscription is processed. However, in some cases, it may take up to two business days to complete the account setup.
