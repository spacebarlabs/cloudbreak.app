Connect Your Apps to FreshRSS

← Back to the FreshRSS product page

Step 1: Create an API Password

Before connecting any mobile or desktop app, you must create a dedicated API password. Your standard web login password will not work for third-party apps.

Direct Link: Go to Settings
  1. Log into your FreshRSS web interface.
  2. Go to Settings (the gear icon) and click on Profile.
  3. Scroll down to the API management section.
  4. Enter a new password in the "API password" field and click submit.

Step 2: Choose Your Device

We've automatically selected your current device, but you can explore apps for other platforms below.

  • iOS & Mac
  • Android
  • Windows
  • Linux

iPhone, iPad, and Mac Setup

NetNewsWire
Recommended for iOS & Mac

Get App

It's free, open-source, and provides a native, seamless reading experience.

  1. Download NetNewsWire from the App Store or Mac App Store.
  2. Open the app and go to settings to Add an Account.
  3. Select FreshRSS (or Google Reader API if FreshRSS isn't explicitly listed).
  4. Enter your CloudBreak server URL: https://freshrss.cloudbreak.app/api/greader.php
  5. Enter your username and the API Password you generated in Step 1.

Android Setup

Capy Reader
Recommended for Android

Get App

It's fast, modern, and built to work perfectly with FreshRSS.

  1. Download Capy Reader from the Google Play Store.
  2. Open the app and add a new account.
  3. Select FreshRSS as your service type.
  4. Enter your CloudBreak server URL: https://freshrss.cloudbreak.app/
  5. Enter your username and the API Password you generated in Step 1.

Windows Setup

Fluent Reader
Recommended for Windows

Get App

A beautiful desktop reading experience designed for modern Windows.

  1. Download Fluent Reader from the Microsoft Store or GitHub.
  2. Go to Settings > Service and select Google Reader API / FreshRSS.
  3. Enter your CloudBreak server URL: https://freshrss.cloudbreak.app/api/greader.php
  4. Enter your username and the API Password you generated in Step 1.

Linux Setup

NewsFlash
Recommended for Linux

Get App

Integrates beautifully with modern Linux desktop environments.

  1. Install NewsFlash via Flathub (Flatpak).
  2. Open the app and choose to add a new FreshRSS or Google Reader account.
  3. Enter your CloudBreak server URL: https://freshrss.cloudbreak.app/api/greader.php
  4. Enter your username and the API Password you generated in Step 1.
